# **WebRTC-server**

This is the WebRTC server to be used with [WebRTC-EveryWhere]() tool as a signalling server. This server, unlike other signalling servers, only uses http web requests.

# Getting Started

## Installation

- Clone this repo

```bash
git clone https://github.com/MadhavGoyal-6/WebRTC-server.git
```

- Set server port in Environment variable `PORT` or use [dotenv](https://www.npmjs.com/package/dotenv)

- Install necessary dependencies

```bash
npm i
```

## Usage

- Just run the server with `npm start`
